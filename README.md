# 🎬 MovieScope – Movie Search and Discovery Web App

MovieScope is a responsive web application that allows users to search, explore, and filter movies using real-time data from a public movie database API.

## 🚀 Quick Start
To view this project:
1. Clone or download the repository.
2. Open `index.html` in your web browser.

## ✨ Features
- **Real-time Movie Data**: Fetches data from the OMDB API.
- **Premium Design**: Modern, dark-themed UI with glassmorphism effects.
- **Responsive**: Works on mobile, tablet, and desktop devices.
- **Simple Logic**: Built using clean `async/await` and `fetch` patterns.

## 🛠 Technologies
- **HTML5**: For structure and semantic markup.
- **Vanilla CSS**: Premium styling without external libraries.
- **JavaScript (ES6+)**: Fetch API for asynchronous communication.
